# Reference Style — ReturnQueen Layout

## Visual Identity
- Background: White / very light gray (#f9fafb)
- Primary accent: Purple-violet (#6B4EFF)
- Text: Dark navy (#111827)
- Muted text: #6b7280
- Cards: White with subtle shadow
- Lots of white space / breathing room

## Layout Sections (top to bottom)
1. **Fixed Nav** — logo left, links center, CTA button right (rounded pill)
2. **Hero** — Split: text left + large image right. Bold headline, bullet points, zip input + button, app store badges
3. **Social proof strip** — "We process returns from 5000+ retailers" + brand logos row
4. **Why section** — Section label, bold H2, 3 feature cards (icon + title + description)
5. **Purple CTA banner** — Full-width purple bg, large text, animated/kinetic words
6. **How it works** — Section label, bold H2, 3 steps with app screenshots
7. **Feature highlight** — Large colored text left (big statement), description paragraph right
8. **Testimonials** — Section label, H2, 3 testimonial cards with stars
9. **App download CTA** — Purple rounded card, phone mockups image, app store buttons
10. **Footer** — Logo + tagline left, 2 link columns (Company, Support), copyright

## Typography
- Headline: Very bold (800-900), large (3-4rem), short and punchy
- Subheadline: Regular, muted gray, 1rem
- Section labels: Small caps, letter-spaced, accent color
- Cards: Medium weight title, light description

## Key Design Patterns
- Rounded corners everywhere (8-16px)
- Subtle box shadows on cards
- Icons in colored rounded squares/circles
- CTA buttons: solid fill, rounded pill shape
- Horizontal dividers between sections
- Max-width container ~1100px centered
